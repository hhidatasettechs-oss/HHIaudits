# HHI Audit Schema

Defines the required structure for governance audit case studies.

## Required Fields
- Audit ID
- Date
- System Evaluated
- Observed Behavior
- Relevant Governance Terms
- Evidence Transcript
- Failure Mode Classification
- Remediation Notes

## Example
Audit ID: HHI-AUDIT-001
Date: YYYY-MM-DD
System Evaluated: Example AI System
Observed Behavior: Governance Drift
Relevant Governance Terms: HHI-GOV-003
Evidence Transcript: See transcript file
Failure Mode Classification: Behavioral Failure
Remediation Notes: Introduce execution-time governance checkpoint
