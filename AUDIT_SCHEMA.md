# HHI Audit Schema

This document defines the structure used for governance audit case studies.

## Required Fields

Audit ID
Date
System Evaluated
Observed Behavior
Relevant Governance Terms
Evidence
Failure Mode Classification
Remediation Notes

## Example

Audit ID: HHI-AUDIT-001
Observed Behavior: Governance Drift
Related Terms: HHI-GOV-003, HHI-AUTH-004
