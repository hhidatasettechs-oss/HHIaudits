# Governance Failure Modes

Standard taxonomy used to classify governance failures observed in HHI audits.

## Authority Failures
- Decision Substitution
- Authority Drift
- Escalation Suppression

## Behavioral Failures
- Governance Drift
- Override Erosion
- Normalization of Workarounds

## Structural Failures
- Accountability Diffusion
- Governance Lag
- Reliance Formation

Each audit should reference at least one category.
