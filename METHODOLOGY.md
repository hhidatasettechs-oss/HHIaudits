# Audit Methodology

This document describes the methodology used to conduct governance audits within the Hollow House Institute ecosystem.

## Method
1. Capture interaction transcript or system behavior.
2. Identify observed governance phenomena.
3. Map observations to HHI governance glossary terms.
4. Classify the failure mode using the governance failure taxonomy.
5. Document remediation implications.

## Objective
The goal of these audits is to document how governance behavior accumulates over time in AI-mediated systems.
